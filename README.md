# projeto_Loca-oCarros
Esta atividade envolve modelagem de dados para uma locadora de carros, com:  

  - Entidades: Cliente, Carro, Agência, Aluguel;
  - Atributos;
  - Relacionamentos;
  - Normalização e DER;
  
Entregas:  

  - Entendimento do Problema;
  - MER e DER;
  - Cardinalidades;
  - Modelo Lógico;
  - Dicionário de Dados;
  - SQL (MySQL);
